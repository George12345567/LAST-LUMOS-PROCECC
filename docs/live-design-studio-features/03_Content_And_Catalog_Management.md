# Category 3: Content And Catalog Management

## Features

1. Content Visibility Toggles (Ratings / Time / Featured)
2. Image Quality Control (Standard / HD)
3. Item Sorting (Name / Price / Rating)
4. Category Filtering
5. Search in items
6. Custom Items Management
7. Add/Edit Item Modal (Image Upload + Featured Toggle)
8. Custom Items Undo/Redo History
9. Favorites (Like/Unlike Items)
