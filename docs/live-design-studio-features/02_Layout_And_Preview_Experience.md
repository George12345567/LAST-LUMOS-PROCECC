# Category 2: Layout And Preview Experience

## Features

1. Layout Controls (Device: Mobile/Tablet/Desktop)
2. View Controls (List / Grid)
3. 3D Preview Controls (Enable + Rotation X/Y)
4. Multi-Page Preview inside device (Home/Menu/Cart/Profile)
5. Template-Aware UI Rendering (Buttons/Categories/Nav/Banners)
6. Featured Banner Variants by template
7. Promo Carousel on Home page
8. Image Zoom Modal
9. Live Analytics Card (Items/Value/Average/Featured)
